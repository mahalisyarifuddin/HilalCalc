# Data Referensi 1400-1900 H & Rumus Linear

## Kriteria Pembuatan Data
Data Referensi (Ground Truth/GT) untuk awal bulan Hijriyah mulai tahun 1400 H hingga 1900 H dihasilkan menggunakan kriteria komposit berikut:
- **Makkah**: Ketinggian Bulan (Altitude) >= 3°, Elongasi >= 6.4° (saat matahari terbenam pada hari ke-29).
- **Kuala Belait (KB)**: Ketinggian Bulan (Altitude) >= 0° (saat matahari terbenam pada hari ke-29).
- **Kondisi**: Kedua set kriteria harus terpenuhi agar bulan baru dimulai pada hari berikutnya. Jika tidak, bulan tersebut digenapkan menjadi 30 hari (istikmal).
- **Tanggal Dasar**: Muharram 1400 H bertepatan dengan JD 2444199 (Selasa, 20 Nov 1979).

## Aproksimasi Rumus Linear
Untuk mendekati kalender Hijriyah menggunakan rumus linear sederhana, rumus berikut telah diturunkan:

```
JD = floor(29.530584908 * Index + 2444199.39) + Hari - 1
```

Dimana:
- `Index = (Tahun - 1400) * 12 + (Bulan - 1)`
- `Bulan` berbasis 1 (1=Muharram, ..., 12=Dzulhijjah).
- `Hari` adalah tanggal dalam bulan Hijriyah tersebut.

## Akurasi
- **Total Bulan**: 6001 (1400 H hingga 1900 H).
- **Kecocokan Tepat (Awal Bulan)**: 4367 (72.77%).
- **Perbandingan**: Rumus ini menawarkan akurasi tinggi (~72.8%) yang setara dengan metode rasional yang lebih kompleks, sambil mempertahankan format linear sederhana dengan offset Epoch yang disesuaikan untuk meminimalkan penyimpangan bulanan.
